# Projeto-Estrutura-de-Dados-Lineares
Projeto acadêmico em C que implementa um sistema de gerenciamento de textos utilizando estruturas de dados fundamentais: pilha encadeada (Undo), fila circular com vetor dinâmico e lista duplamente encadeada ordenada. O projeto utiliza alocação dinâmica de memória (malloc, realloc e free) e organização modular com arquivos .h e .c.
